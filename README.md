# Informe Ejecutivo · Conciliaciones de Inventario de Tinas 2025–2026

Dashboard gerencial estático que consolida y analiza la información operativa
del proceso de conciliación de tinas para el comité gerencial.

🔗 **Demo en vivo**: https://msduran1994-hash.github.io/informe-tinas-conciliaciones/

## Fuente de datos
- Conciliaciones Inventario Tinas 2025.xlsx · hoja `CONSOLIDADO TINAS`
- Conciliaciones Inventario Tinas 2026.xlsx · hoja `CONSOLIDADO TINAS`
- 299 registros consolidados (deduplicados por informe + bodega + corte)

## Módulos
KPIs · Alertas automáticas · Comparativo 2025/2026 · Tendencias ·
Ranking por sede · Heatmap · Hallazgos priorizados · Plan de acción ·
Diagnóstico técnico/financiero · Detalle trazable

## Stack
HTML + CSS + JS vanilla. Sin build. Sin dependencias.
